package com.jdblabs.jlp;

import com.jdblabs.jlp.ast.SourceFile;

JLPParser is a simple interface. It has one method to return a parsed SourceFile given an input string. It may be expanded in the future to be an abstract class implementing methods that take additional input for convenience.

 
public interface JLPParser {
    public SourceFile parse(String input); }
