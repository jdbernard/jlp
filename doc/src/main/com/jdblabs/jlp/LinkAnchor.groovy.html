package com.jdblabs.jlp

import com.jdblabs.jlp.ast.ASTNode

A LinkAnchor in the documentation is very similar to an HTML anchor. It creates a reference in the documentation that can used by the author to link to this point.

The author uses a URL with the jlp protocol to refer to anchors. For example: jlp://jlp.jdb-labs.com/LinkAnchor refers to this documentation.

LinkAnchor is a data class to hold information about the link anchors defined by @org directives.

 
public class LinkAnchor {

    public enum LinkType { InlineLink, BlockLink, FileLink }

The anchor id. This comes from the text after the directive.

 
    public String id

    public LinkType type = LinkType.BlockLink
    public ASTNode source
    public String sourceDocId

}
